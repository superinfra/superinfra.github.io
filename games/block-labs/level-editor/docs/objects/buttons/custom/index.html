Custom Button

Custom Buttons are triggered when a block, player, or NPC collides with the button. After they're triggered, the builder-written code inside runs!

CAUTION: For use by Trusted Builders only. Your level will be automatically rejected if you use this object while not a Trusted Builder.

Usage

Expand the button in the Explorer window (View > Explorer) and double-click the Script inside. It should have a section like this: 

if touch:FindFirstChild("IsBox") then
    print("A block pressed the button!")
    button.Material = Enum.Material.Neon
    activated = true
elseif humanoid and plr then
    print("A player pressed the button!")
    button.Material = Enum.Material.Neon
    activated = true
elseif humanoid then
    print("An NPC pressed the button!")
    button.Material = Enum.Material.Neon
    activated = true
end

For block activation:

Go to the first indent of the section shown above and enter your code. The variable touch references the block that activated the button.

For player activation:

Go to the second indent of the section shown above and enter your code. The variable plr references the player that activated the button, and humanoid references that player's character's humanoid. touch.Parent, humanoid.Parent, and plr.Character all reference the player's character.

For NPC activation:

Go to the third and final indent of the section shown above and enter your code. The variable humanoid references the triggering NPC's humanoid. Both touch.Parent and humanoid.Parent reference the actual NPC.

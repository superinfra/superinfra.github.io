Teleporter

A Teleporter teleports Blocks from its Start part to its End part. It can also teleport players if you enable AcceptPlayers.

Usage

When the teleporter's Start is touched by a block, it teleports to the End. It can also work the other way around if you disable OneWay.

Properties

Property Description Acceptable Values
OneWay If true, End can teleport players and blocks to Start. True or false
Instant If false, the teleporting animation won't be played. (Only affects players) True or false
AcceptPlayers Whether or not players can use the teleporter True or false

Tips

  • Two way teleporters should only be used on small and/or round parts. Otherwise, the player may not be able to get off in time, causing a teleport loop.

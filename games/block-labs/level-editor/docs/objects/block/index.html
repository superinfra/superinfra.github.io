Block

Blocks, the most important objects in the game, are pushable objects used to solve puzzles, press buttons, cross gaps, etc. They are spawned by Block Spawners and destroyed by Block Destroyers, but they can also be placed directly into a level.

Usage

You can keep Blocks into ReplicatedStorage's Blocks folder to spawn them with Block Spawners, or you can place them in Workspace to spawn them directly into the enviroment.

A pack of basic blocks comes with your level kit in the Blocks folder, but you can read Creating Custom Blocks to learn how to make your own shapes and types.

Properties

Property Description Acceptable Values
IsBox Used internally True
CAUTION: The IsBox property is how buttons detect blocks. Do not set it to false or delete it.

Touch Block Spawner

You may be looking for Block Spawner.

A Touch Block Spawner, like a normal Block Spawner, spawns blocks when you trigger them. The difference is that Touch Block Spawners need to be touched by the player's avatar instead of being clicked.

Usage

When its button is pressed, it will spawn the block specified in the BlockToSpawn property on top of the "Block Spawn" text. It'll also delete any blocks created by spawners with the same SpawnerID, including itself and regular Block Spawners.

Properties

Property Acceptable Values
BlockToSpawn The name of any block
SpawnerID Any text (typically a number)

Tips

  • Change the SpawnerID property to create multiple blocks at the same time!
